残差モデルの実装

現在
scripts/run_train_expert_blending_8experts_v4_paired_uniform50_noise0.sh
で学習。
内部のNNUEモデルの合成は、以下のようになっている。
初期化:
B: ベースラインNNUE重み
C_1, C_2, ..., C_8: 学習対象になるNNUE重み
C_1 <- B
C_2 <- B
...
C_8 <- B
学習:
合成される重みW=C_1*w_1+C_2*w_2+...+C_8*w_8

新しく実装したい、残差モデルバージョン
初期化:
B: ベースラインNNUE重み
C_1, C_2, ..., C_8: 学習対象になるNNUE重み
C_1 <- 0
C_2 <- 0
...
C_8 <- 0
学習:
合成される重みW=B+C_1*w_1+C_2*w_2+...+C_8*w_8
