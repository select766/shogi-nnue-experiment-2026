# コンピュータ将棋: nnue-pytorchによるNNUE評価関数の学習の環境構築

目標: 棋譜データセットファイルを入力とし、NNUE評価関数を学習する環境構築をしたい。

gitで必要なリポジトリをcloneし、ビルド、教師あり学習を実行せよ。
学習するモデルは、標準NNUE形式とする。
Pythonについては、グローバルにパッケージをインストールせず、uvを用いて、妥当なpythonバージョンの仮想環境を作成せよ。

本環境はUbuntu 24.04で、NVIDIA GPUが存在する。GPUを利用した学習ができるようにすること。
成功し次第、環境構築の方法を、 `how-to-setup-nnue-pytorch.md` としてドキュメント化せよ。

# データセット
`subset_tanuki-.nnue-pytorch-2024-07-30.1` ディレクトリに、大きなデータセットのサブセットが存在する。
これを用いて環境構築の成功を確認する。train/valに分けてシャッフルする必要があり、その環境構築も必要。
